# Imagem de exemplo no GitHub

## README do repositório (o que aparece em github.com/Square2-Inc/AdaptiveCaret)

1. Copia o teu ficheiro **Example.png** para a pasta **`docs/`** deste projeto.
2. Renomeia para **`example.png`** (recomendado; o README já aponta para `./docs/example.png`).
   - No Windows podes manter **Example.png** e alterar no [README.md](../README.md) a linha da imagem para `./docs/Example.png`.
3. Na raiz do projeto:

```powershell
git add docs/example.png
git commit -m "docs: imagem de exemplo Adaptive Caret"
git push origin main
```

Depois do push, o README mostra a captura automaticamente.

---

## Pré-visualização ao partilhar o link (opcional)

Quando colas o URL do repo no Slack, LinkedIn, etc., o GitHub pode mostrar uma imagem grande:

1. No GitHub: **Settings** do repositório → **General**.
2. Secção **Social preview** → **Edit** → carrega uma imagem (recomendado **1280×640** px).
3. Podes reutilizar a mesma **`example.png`** ou um banner da Square².

Isto é independente da imagem no README (são dois sítios diferentes).

---

## Perfil pessoal do GitHub (opcional)

O **perfil** `github.com/teu-utilizador` usa um repositório especial com o **mesmo nome** que o teu username (`username/username`). Se quiseres banner lá, é outro fluxo; para **este projeto**, basta o README acima.
